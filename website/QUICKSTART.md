# 🚀 Quick Start - Webapp Admin

## En 3 Étapes

### 1️⃣ Lancer le Serveur Web
```bash
cd website
./run.sh
```
➜ Accédez à **http://localhost:8787**

### 2️⃣ Configurer l'API
1. Trouvez le champ **API** en haut à gauche
2. Entrez l'URL: `http://localhost:8000/api` (ajustez le port selon votre setup)
3. Cliquez **Sauver**

### 3️⃣ Se Connecter
1. Entrez vos identifiants (username/email + password)
2. Cliquez **Se connecter**
3. Le dashboard devrait afficher les données ✨

---

## ✅ C'est Fait!

Si tout fonctionne:
- ✅ Vous verrez les statistiques du dashboard
- ✅ Les onglets Chat/Events/Utilisateurs seront accessibles
- ✅ Les données s'afficheront sans erreur

---

## 🐛 Problème?

### Erreur "Connection error"?
- Vérifiez que votre API backend est lancée
- Vérifiez l'URL dans le champ API
- Ouvrez la console (F12) pour voir les logs détaillés

### Erreur "401 Unauthorized"?
- Vérifiez vos identifiants
- Essayez de vous reconnecter

### Les données ne s'affichent pas?
- Ouvrez la console (F12)
- Cherchez les messages d'erreur en rouge
- Consultez le guide détaillé: `website/SETUP.md`
